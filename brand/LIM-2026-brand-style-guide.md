# **2026 Brand Style Guide**

This Comprehensive Brand Style Guide establishes the definitive visual and strategic standards for **Lutheran Indian Ministries (LIM)**. It integrates the organization’s "Native-Led" identity with modern **Trauma-Informed Design (TID)** and high-performance digital standards.

---

## **1\. Brand Essence & Core Philosophy**

### **1.1 The Theological Foundation**

The brand is anchored in the **"Christ, the Bridge"** philosophy, which visually and narratively connects Native American iconography with Christian symbols.

* **Mission:** Proclaiming the Gospel, discipling Native leaders, and healing historical trauma.  
* **Three Pillars:** The organizational structure is built on **Proclaim** (cultural Gospel sharing), **Disciple** (leadership development like Haskell LIGHT), and **Heal** (trauma and addiction recovery).

  ### **1.2 Design Ethos: Trauma-Informed Design (TID)**

The website and marketing materials must function as a **"Safe Harbor"** for seekers.

* **Safety & Predictability:** Layouts must be consistent and low-arousal, avoiding jarring pop-ups or high-stress visual contrasts.  
* **Empowerment:** Content must include warnings for sensitive topics and "Quick Exit" strategies for users in unsafe physical environments.  
  ---

  ## **2\. Visual Identity: The Logo**

  ### **2.1 Primary Mark (Horizontal Stacked)**

The **Horizontal Stacked** version is the official primary logo treatment. It provides the best balance for the long organizational name and ensures the brand mark remains prominent.

* **Typeface:** **Barlow Condensed** (Bold/ExtraBold).  
* **Color:** Use **Charcoal (\#282924)** for all text to provide a softer, more premium look than pure black.  
* **Alignment (Pixel-Perfect Rule):** The top of the letter "L" must align with the top edge of the mark, and the baseline of "MINISTRIES" must align with the bottom edge of the mark to create a unified block.  
  ---

  ## **3\. Official Color Palette**

The palette is divided into **Core Identity** colors and **Atmospheric Buffers** to ensure a balanced user experience.

### **3.1 Core Medicine Wheel Colors**

Used for primary branding and high-priority actions.

| Color | Hex Code | Usage |

| **LIM Red** | **\#ED1C24** | **Action:** Use for "Donate" buttons or critical "Need Help?" alerts. |
| **LIM Yellow** | **\#FFDE16** | **Highlight:** Use for accents, secondary buttons, and hope-focused graphics. |
| **LIM Black** | **\#000000** | **Identity:** Used only within the brand mark for maximum strength. |

Used as background washes to create a calming, natural environment.

| Color | Hex Code | Vibe | Usage |

| Color | Hex Code | Usage |

---

## **4\. Typography Hierarchy**

Consistent typography reinforces the bridge between tradition and modern mission.

* **Headings (Primary):** **Merriweather** or **Playfair Display**. Serif fonts convey theological depth and warmth.  
* **Headings (Secondary/UI):** **Barlow Condensed**. Used for all-caps subheads and button text for a modern, "action-oriented" feel.  
* **Body Text:** **Open Sans** or **Lato**. Clean sans-serif fonts ensure accessibility for elders and mobile users.  
  ---

  ## **5\. Digital & UX Standards**

  ### **5.1 Navigation Strategy**

The site must employ a **Dual-Audience path**.

1. **Seeker Path:** Prioritize "Find Healing" and "Start Your Journey" with Soft Sky Blue/Sage Green accents.  
2. **Donor Path:** Prioritize "Fuel the Mission" with high-contrast LIM Red buttons.

   ### **5.2 Imagery Standards**

* **Authenticity:** Use only original photography of Native elders, Haskell students, and actual healing circles.  
* **Visual Tone:** Photos should have a "warm" color grade and avoid high-arousal or exploitative "trauma porn".  
* **Graphic Elements:** Incorporate the **Circle motif** (Medicine Wheel) as a recurring design element for frames and iconography.

  ### **5.3 Technical Essentials**

* **Donation:** Must feature embedded giving (no redirects) and support Apple/Google Pay.  
* **Transparency:** Prominently display the **3-Star Charity Navigator** and **GuideStar Platinum** seals in the footer.
